## 1. Project Overview

### What did you build?

I built a multi-page website called **Sports Hub** that provides information about popular sports including football, basketball, and sports training. The website is designed to educate users about each sport, famous athletes, and the importance of physical fitness.

The site includes five interconnected pages: a homepage, football page, basketball page, training page, and contact page. Each page contains structured content, images, tables, and navigation links.

The design is clean, responsive, and focuses on easy access to sports information for general users.

---

### How do the pages work together?

The homepage introduces Sports Hub and links to the three main sections: football, basketball, and training. Each sport has its own dedicated page with detailed information.

The navigation bar is consistent across all pages, allowing users to move easily between sections. The contact page allows users to send messages or feedback.

## 3. Technical Implementation

### HTML

- Semantic structure used (header, nav, main, section, footer)
- Tables used for sports events and training schedules
- Forms used in contact page (name, email, subject, message)
- Lists used (ordered and unordered) across all pages
- Images include descriptive alt text for accessibility
- Internal navigation links connect all pages
- External links to ESPN, FIFA, NBA, Healthline

---

### CSS

- External stylesheet used (styles.css)
- Organized into sections (layout, grid, forms, responsive)
- Consistent color scheme (blue, dark grey, white, orange accents)

---

### Flexbox Usage

- Navigation bar uses Flexbox for horizontal layout
- Cards and layout elements adapt using flex-wrap for responsiveness

---

### CSS Grid Usage

- Homepage sports section uses Grid (3-column layout)
- Automatically adapts to 1 column on mobile

---

### Responsive Design

- Mobile-first responsive design using media queries
- Layout changes at 768px breakpoint
- Navigation stacks vertically on small screens
- Grid changes from 3 columns to 1 column

---
