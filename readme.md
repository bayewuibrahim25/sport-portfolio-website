
# Sports Hub Website

## Overview

This project is a multi-page sports website developed using HTML5 and CSS3. The website provides information about football, basketball, and sports training.

The project demonstrates modern web development techniques including responsive design, semantic HTML, and structured CSS styling.

---

## Website Pages

Home – introduction to the website  
Football – information about football and player statistics  
Basketball – information about basketball and famous players  
Training – sports training and weekly exercise schedule  
Contact – contact form and external links

---

## Features

The website includes:

- Multi-page navigation system
- Tables displaying sports information
- Lists of players and sports
- Images with alternative text
- Internal links between pages
- External links to sports websites
- Contact form
- Responsive design
- CSS Flexbox layout
- CSS Grid layout

---
