## 1. Regular Maintenance Tasks

1. **Check all navigation links weekly** – ensures all pages still connect correctly  
2. **Test website on mobile monthly** – ensures responsive design still works  
3. **Check images load properly monthly** – prevents broken images  
4. **Update sports information every 3–6 months** – keeps content accurate  
5. **Test form functionality monthly** – ensures contact form works correctly  
6. **Backup files after major updates** – prevents data loss  

---

## 2. Updating Content

### Adding new content
1. Copy existing HTML structure from similar page  
2. Create new section or page  
3. Add navigation link to all pages  
4. Add images to images folder  
5. Test layout  
6. Commit and push to GitHub  

---

### Updating content
1. Open correct HTML file  
2. Edit text or images  
3. Save changes  
4. Test in browser  
5. Push to GitHub  

---

### Removing content
1. Delete section from HTML  
2. Remove related navigation links  
3. Check no broken links remain  
4. Update homepage if needed  

---

## 3. Technical Maintenance

### If a page breaks:
1. Check browser console (F12)  
2. Validate HTML structure  
3. Check file paths for images  
4. Revert to last working version using Git  

---

### Design changes:
- Update CSS file
- Test all pages after changes
- Ensure mobile layout still works

---

### Adding new pages:
1. Create new HTML file  
2. Copy structure from existing page  
3. Add to navigation on all pages  
4. Test links  
5. Push to GitHub  

---

## 4. Website Health Checks

### Performance
- Pages load quickly
- Images are optimized
- No large unused files

### Accessibility
- All images have alt text
- Links are readable
- Text contrast is clear

### Standards
- HTML validated regularly
- CSS checked for errors
- Works on Chrome, Firefox, Safari

## 6. Common Problems

**Problem:** Image not showing  
Solution: Check file name and folder path  

**Problem:** Navigation broken  
Solution: Check link spelling in all pages  

**Problem:** Layout broken on mobile  
Solution: Check media queries in CSS
