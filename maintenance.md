# Website Maintenance Guide

## Website Structure

The website contains five HTML pages connected by a navigation menu.

Files include:

index.html  
football.html  
basketball.html  
training.html  
contact.html  

styles.css  

images folder for image files

---

## Updating Text

To update text content:

1. Open the relevant HTML file
2. Edit the text between the HTML tags
3. Save the file and refresh the browser

---

## Adding a New Page

To add a new page:

1. Create a new HTML file
2. Copy the structure from an existing page
3. Update the title and content
4. Add the new page link to the navigation menu

---

## Adding Images

Images are stored in the **images folder**.
